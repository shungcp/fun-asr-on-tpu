# fun-asr-on-tpu

This is a hobby project to try debugging and running ASR model inference on Trillium(v6e) and Ironwood(late 2026).

这是一个个人爱好者项目，欢迎有兴趣移植和优化的同学加入。

目的是为了调试和运行一些流行ASR/TTS模型在TPU上的推理任务。

当然也会在移植的过程中，收获学习和成长的乐趣。
